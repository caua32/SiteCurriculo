Sobre o Projeto

Este site foi desenvolvido como parte de um projeto realizado durante o programa Jovem Aprendiz. A NerdFlix é uma plataforma de votação voltada para cinéfilos, inspirada no Letterboxd. O site conta com múltiplas abas de votação e foi projetado para oferecer uma navegação intuitiva, priorizando a facilidade de uso.
